title: Shell Integration

clim provides two shell integration features: **completions** for tab completion and **hooks** for automatic `.clim.yaml` checking.

## Tab Completion

Generate native tab completions for all clim commands, flags, and arguments.

### Setup

import { Tabs, TabItem } from '@astrojs/starlight/components';

<Tabs>
<TabItem label="bash">
```bash
# Current session
source <(clim shell completion bash)

# Persistent — add to ~/.bashrc
echo 'source <(clim shell completion bash)' >> ~/.bashrc
```
</TabItem>
<TabItem label="zsh">
```bash
# Current session
source <(clim shell completion zsh)

# Persistent — add to ~/.zshrc
echo 'source <(clim shell completion zsh)' >> ~/.zshrc
```
</TabItem>
<TabItem label="fish">
```bash
# Current session
clim shell completion fish | source

# Persistent
clim shell completion fish > ~/.config/fish/completions/clim.fish
```
</TabItem>
<TabItem label="PowerShell">
```powershell
# Current session
clim shell completion powershell | Out-String | Invoke-Expression

# Persistent — add to $PROFILE
Add-Content $PROFILE 'clim shell completion powershell | Out-String | Invoke-Expression'
```
</TabItem>
</Tabs>

## Shell Hook

The shell hook automatically checks `.clim.yaml` requirements when you `cd` into a project directory — like nvm or direnv for your toolchain.

### How It Works

1. You `cd` into a directory
2. The hook walks up the directory tree looking for `.clim.yaml`
3. If found, it runs `clim check` silently
4. Only prints output when tools are missing or outdated

### Setup

<Tabs>
<TabItem label="bash">
```bash
eval "$(clim shell hook bash)"

# Persistent — add to ~/.bashrc
echo 'eval "$(clim shell hook bash)"' >> ~/.bashrc
```
</TabItem>
<TabItem label="zsh">
```bash
eval "$(clim shell hook zsh)"

# Persistent — add to ~/.zshrc
echo 'eval "$(clim shell hook zsh)"' >> ~/.zshrc
```
</TabItem>
<TabItem label="fish">
```bash
clim shell hook fish | source

# Persistent
clim shell hook fish > ~/.config/fish/conf.d/clim-hook.fish
```
</TabItem>
<TabItem label="PowerShell">
```powershell
clim shell hook powershell | Out-String | Invoke-Expression

# Persistent — add to $PROFILE
Add-Content $PROFILE 'clim shell hook powershell | Out-String | Invoke-Expression'
```
</TabItem>
</Tabs>

### Example

When you `cd` into a project with missing or outdated tools:

```
$ cd my-k8s-project
    ✗ kubectl              —            (>=1.28)
    ⚠ terraform            1.5.0        (>=1.7)
  Run 'clim check' for details or 'clim import' to install missing tools.
```

## Both Together

For the best experience, set up both completion and hook:

```bash
# bash
echo 'source <(clim shell completion bash)' >> ~/.bashrc
echo 'eval "$(clim shell hook bash)"' >> ~/.bashrc
```
